# My Dotfiles

## Install / Bootstrap
1. install bash from homebrew
1. Follow steps in https://www.shell-tips.com/mac/upgrade-bash/
1. run `./bootstrap`
1. install tmux
1. install fzf 
  1. run $(brew --prefix)/opt/fzf/install
1. install tmux
1. install git@github.com:erikw/tmux-powerline.git in ~/.tmux/
